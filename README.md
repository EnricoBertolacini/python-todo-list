# To-Do List em Python

Este é um projeto simples de lista de tarefas feito em Python para praticar lógica de programação.

## Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Marcar tarefas como concluídas
- Remover tarefas

## Tecnologias utilizadas

- Python

## Objetivo do projeto

Este projeto foi desenvolvido para praticar conceitos fundamentais de programação como:

- Listas
- Dicionários
- Funções
- Estruturas de repetição
- Condicionais
- Tratamento de exceções
